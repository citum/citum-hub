# Citation Style Editor Client

This is the SvelteKit-based frontend for the Citum Hub.

## Tech Stack

- **Framework**: Svelte 5 (with SvelteKit 2)
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide Svelte & Material Symbols
- **Type Checking**: TypeScript

## Development

Run the development server:

```bash
npm run dev
# OR
bun run dev
```

## Build

Build the production-ready application:

```bash
npm run build
# OR
bun run build
```

## Check

Run type and svelte checking:

```bash
npm run check
# OR
bun run check
```
